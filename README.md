# Introduction
This script converts a csv file of translations into iOS Localizable.strings files

# Usage

* Convert csv to localizable strings files:

`./csv2strings.sh <csvFile> <stringsfile1> <stringsfile2>`

# CSV File format

Semicolon ";" separated CSV
`key;lang1;lang2;lang3`

Skip with `//`
